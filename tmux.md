# tmux

## Bind-key

`Ctrl+A`

## Show commands

`Ctrl+A ?`

## Detach

`Ctrl+A d`

## Rename

`Ctrl+A ,`

## New pane

`Ctrl+A %`

## Switch between split modes

`Ctrl+A Space`

## Switch between split window

`Ctrl+A <arrow>`

## Create new window

`Ctrl+A c`

## Switch between windows

`Ctrl+A n` or `Ctrl+p`


## Find window

`Ctrl+A f`

## Move pane to new window

`Ctrl+A !`

## Kill pane

`Ctrl+A :kill-window`

## Copy paste

`Ctrl+A [`

### Select text

Space
